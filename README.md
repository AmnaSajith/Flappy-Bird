# Flappy Bird Game

A simple **Flappy Bird clone** built using **HTML, CSS, and JavaScript**.
The game runs directly in the browser using the **HTML5 Canvas API** and implements basic game physics, obstacle generation, and collision detection.

---

## 🚀 Features

* Smooth bird movement with gravity and jump mechanics
* Randomly generated pipes for obstacles
* Real-time score tracking
* Collision detection between bird and pipes
* Game over and restart functionality
* Lightweight browser-based game

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the web page
* **CSS3** – Styling and background
* **JavaScript** – Game logic and physics
* **Canvas API** – Rendering graphics

## 🎮 How to Play

1. Open `index.html` in your browser.
2. Press **Space / Arrow Up / X** to make the bird jump.
3. Avoid hitting the pipes.
4. Pass through pipes to increase your score.
5. If you collide with a pipe or fall, the game ends.

Press the jump key again to **restart the game**.
